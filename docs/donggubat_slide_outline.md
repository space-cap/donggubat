# 🎤 동구밭 브랜드 분석 발표 슬라이드 구성안

> **발표 목적**: 마케팅 수업에서 동구밭을 모르는 청중에게 브랜드를 분석·소개
> **슬라이드 수**: 10장
> **톤**: 깔끔하고 전문적인 비즈니스 톤 + 데이터 중심
> **관점**: 객관적 브랜드 분석 (개인 경험 제외)
> **제작 도구**: 구글 슬라이드 & 감마(Gamma)로 각각 제작 후 비교

---

## 📐 슬라이드 구성 개요

```
 1️⃣ 표지

 5️⃣ 제품력 심화 — 샴푸바

 8️⃣ 구매 페르소나 — 샴푸바

 🔟 핵심 인사이트 & 결론
```

## 📄 슬라이드별 상세 구성

---

### 슬라이드 1️⃣ — 표지

**제목 (택 1)**:
- `"먼저 좋아서 사고, 알고 보니 더 좋은 곳이었다" — 동구밭 브랜드 분석`
- `동구밭(DONGGUBAT) — 제품력으로 증명하는 사회적기업`
- `고체 비누 하나로 시장을 바꾸다 — 동구밭 브랜드 분석`

**포함 요소**:
- 동구밭 로고 또는 대표 제품 이미지
- 부제: 친환경 고체 생활용품 브랜드 분석
- 발표자 이름, 수업명, 날짜

**디자인 팁**:
- 동구밭 브랜드 컬러(어스톤, 그린, 베이지) 활용
- 깔끔한 배경 + 제품 사진 한 장이면 충분
- 너무 많은 정보를 넣지 말 것 (제목 + 부제 + 이름만)

---

### 슬라이드 2️⃣ — 동구밭은 누구인가?

**핵심 메시지**: "마을 어귀의 작은 텃밭에서 시작된 브랜드"

**포함 요소**:

| 항목 | 내용 |
|------|------|
| 설립 | 2017년, 사회적기업 |
| 창업자 | 노순호 대표 |
| 이름 뜻 | 동구(마을 어귀) + 밭(텃밭) |
| 미션 | 발달장애인과 비장애인이 함께 만드는 친환경 생활용품 |
| 핵심 가치 | 공존 · 자연 · 진정성 |
| 직원 구성 | 전체 직원의 약 60%가 발달장애인 (가꿈지기) |

**발표 포인트**:
- 동구밭의 뜻을 설명하면 청중이 브랜드를 기억하기 쉬움
- "장애인 고용 = 착한 소비"가 아니라, "함께 일하는 것이 일상"이라는 관점 강조

**디자인 팁**:
- 타임라인이나 브랜드 스토리 흐름으로 시각화
- 동구밭 작업 현장 사진 활용 가능

---

### 슬라이드 3️⃣ — 시장 분석

**핵심 메시지**: "친환경 생활용품 시장은 빠르게 성장 중"

**포함 데이터**:

| 데이터 | 수치 |
|--------|------|
| 국내 친환경 생활용품 시장 | 약 2조 원 규모 (2024 기준) |
| 연평균 성장률 (CAGR) | 8~12% |
| 고체 샴푸 시장 성장률 | 연 15% 이상 급성장 |
| MZ세대 친환경 제품 구매 의향 | 72% 이상 |
| 제로 웨이스트 관심도 | 최근 5년간 검색량 300% 증가 |

**경쟁 환경 (간략)**:

| 브랜드 | 특징 | 포지션 |
|--------|------|--------|
| 동구밭 | 사회적기업 + 제품력 | 가치 + 품질 |
| 러쉬(LUSH) | 글로벌 프리미엄 | 프리미엄 체험 |
| 닥터브로너스 | 유기농 인증 | 성분 중심 |
| 아로마티카 | 국내 클린뷰티 | 성분 + 디자인 |
| 마켓컬리 PB | 유통채널 PB | 가성비 |

**디자인 팁**:
- 그래프/차트 활용 (막대 그래프 or 도넛 차트)
- 경쟁 포지셔닝 맵 (가격-가치 2x2 매트릭스) 삽입하면 임팩트 있음
- 수치에는 반드시 출처를 작게 표기

---

### 슬라이드 4️⃣ — 무엇을 파는가? (제품 라인업)

**핵심 메시지**: "고체 비누 하나에 이렇게 많은 카테고리가?"

**제품 카테고리 정리**:

```
🧴 퍼스널 케어          🍽️ 주방/생활          🎁 기타
──────────────         ──────────────       ──────────
샴푸바 (3종)            설거지바              선물세트
린스바                  다목적 세탁바          비누 받침대
바디워시바              주방비누              거품망
세안바                                      트래블 키트
쉐이빙바
```

**핵심 수치 (있으면 좋은 것들)**:
- 총 SKU 수
- 베스트셀러 제품
- 누적 판매량 (있다면)
- 주요 판매 채널 (자사몰, 마켓컬리, 29CM, 아이디어스 등)

**디자인 팁**:
- 제품 사진을 카테고리별로 배치
- 아이콘 활용하면 깔끔
- 너무 많은 제품을 나열하지 말고 대표 제품 중심으로

---

### 슬라이드 5️⃣ — 제품력 심화: 샴푸바 🧴

**핵심 메시지**: "고체라서 불편한 게 아니라, 고체라서 더 좋다"

**포함 요소**:

| 항목 | 내용 |
|------|------|
| 라인업 | 중건성 / 지성 / 쿨링 — 두피 타입별 맞춤 |
| 핵심 성분 | 코코넛 추출 천연 계면활성제, 식물 유래 성분 |
| 인증 | 비건 인증 |
| 세정력 | 거품 풍성, 액상 샴푸 대비 동등 이상의 세정력 |
| 패키징 | 플라스틱 제로 — 종이 포장 + 점자 표기 |

**고객 리뷰 데이터 (정량적)**:
- 화해 앱 평점: ★ 4.2 이상
- 주요 긍정 키워드: "거품 좋음", "두피 개운", "성분 순함"
- 주요 보완 키워드: "처음에 뻣뻣" → 린스바 병행 시 해결

**디자인 팁**:
- 제품 사진 + 핵심 스펙을 카드형으로 배치
- 리뷰 키워드를 워드클라우드로 시각화하면 효과적
- Before(플라스틱 통) vs After(고체바) 비교 이미지

---

### 슬라이드 6️⃣ — 제품력 심화: 설거지바 🍽️

**핵심 메시지**: "1종 주방세제 인증, 아기 식기까지 안심"

**포함 요소**:

| 항목 | 내용 |
|------|------|
| 인증 | **1종 주방세제** (채소·과일·아기 식기 세척 가능) |
| 핵심 성분 | 코코넛오일, 올리브유, 베이킹소다 (기름기 제거) |
| 세정력 | 뽀득뽀득 세정감, 헹굼이 빠르고 잔여물 적음 |
| 차별점 | 맨손 설거지해도 손 건조함 적음 |
| 패키징 | 플라스틱 용기 제로 |

**고객 리뷰 데이터 (정량적)**:
- 주요 긍정 키워드: "뽀득뽀득", "손 안 거칠어짐", "잔여물 없음"
- 주요 보완 키워드: "강한 기름기는 뜨거운 물로 애벌 헹굼 필요"
- 재구매율: 높음 (정기 구매 고객 비중 높음)

**디자인 팁**:
- 샴푸바 슬라이드와 동일한 레이아웃으로 통일감
- 액체 세제 vs 설거지바 비교표 넣으면 효과적
- "1종 주방세제" 인증 마크 강조

---

### 슬라이드 7️⃣ — 브랜드 포지셔닝

**핵심 메시지**: "착한 브랜드가 아니라, 실력 있는 브랜드"

**포함 요소**:

#### 포지셔닝 맵 (2x2 매트릭스)

```
                 고가격
                   │
     러쉬(LUSH)    │    닥터브로너스
                   │
 ─── 감성/체험 ────┼──── 기능/성분 ───
                   │
     마켓컬리PB     │    ★ 동구밭
     대형마트 PB    │    아로마티카
                   │
                 저가격
```

#### 동구밭만의 독보적 포지션

| 차별 요소 | 설명 |
|-----------|------|
| 🏭 사회적기업 | 직원 60%가 발달장애인 → 고용 자체가 미션 |
| 🧼 검증된 제품력 | "착해서"가 아니라 "좋아서" 재구매 |
| ♻️ 진짜 제로 웨이스트 | 제품·포장·물류까지 플라스틱 프리 |
| 🔤 포용적 디자인 | 패키지에 점자 표기 |
| 💰 합리적 가격대 | 프리미엄 대비 접근성 높은 가격 |

**디자인 팁**:
- 포지셔닝 맵을 깔끔한 차트로 시각화 — 이 슬라이드의 핵심!
- 동구밭 위치를 별(★)이나 색상으로 강조
- 차별 요소는 아이콘과 함께 간결하게

**디자인 팁**:

### 슬라이드 🔟 — 핵심 인사이트 & 결론

**핵심 메시지**: 동구밭이 마케팅에서 가르쳐주는 것

#### 인사이트 1: 구매 동기의 진짜 구조

```
❌ 기존 프레임:  "착한 기업" → 동정 구매 → 1회성

✅ 동구밭의 현실: 제품력 감동 → 재구매 → 가치 발견 → 충성 고객
```

#### 인사이트 2: 핵심 공식

```
┌─────────────────────────────────────┐
│                                     │
│   제품력 (Product) × 가치 (Value)    │
│   = 지속 가능한 브랜드 충성도         │
│                                     │
│   · 제품력만 있으면 → 대체 가능       │
│   · 가치만 있으면 → 동정 소비, 1회성  │
│   · 둘 다 있으면 → 동구밭 ★          │
│                                     │
└─────────────────────────────────────┘
```

#### 인사이트 3: 마케팅 시사점 (한 줄 요약)

> **"먼저 좋아서 사고, 알고 보니 더 좋은 곳이었다."**
> — 이것이 사회적기업 마케팅의 가장 이상적인 구조다.

**디자인 팁**:
- 큰 텍스트로 핵심 메시지를 강렬하게
- 시각적 공식(제품력 × 가치 = 충성도)을 도식화
- 마지막 한 줄은 인용구 형태로 임팩트 있게
- Q&A 안내 또는 "감사합니다" 추가

### 레이아웃 원칙

- **여백을 넉넉하게** — 슬라이드가 빽빽하면 청중이 읽지 않음
- **한 슬라이드 = 한 메시지** — 10초 안에 핵심이 파악되어야 함
- **텍스트는 최소화** — 발표에서 말로 보충, 슬라이드는 시각 자료
- **차트/표/아이콘** 활용 — 텍스트보다 시각 자료가 2배 효과적
- **통일된 레이아웃** — 같은 유형의 슬라이드는 같은 템플릿 사용
