# 🧴 동구밭 제품별 구매 페르소나

> **핵심 전제**: 동구밭 고객은 장애인 고용이나 친환경 가치 때문에 "착한 소비"를 하는 것이 아니다.
> **제품이 진짜 좋아서 먼저 구매한다.** 사회적 가치는 구매 후에 알게 되는 "기분 좋은 보너스"다.

---

## 🔑 구매 동기의 우선순위 재정립

기존의 동구밭 브랜드 분석에서는 "사회적 가치"와 "친환경"을 앞세웠지만,
실제 구매 현장에서 고객의 마음은 다음과 같은 순서로 움직입니다:

```
기존 인식 (바깥에서 본 시각)
━━━━━━━━━━━━━━━━━━━━━━━━━━━
 1순위: 🌍 사회적 가치 (장애인 고용)
 2순위: 🌿 친환경 (제로 웨이스트)
 3순위: 🧼 제품 품질

실제 구매 심리 (고객의 진짜 마음)
━━━━━━━━━━━━━━━━━━━━━━━━━━━
 1순위: 🧼 "이 제품 진짜 좋다"
 2순위: 💡 "성분이 순하고 안심된다"
 3순위: 😊 "게다가 환경에도 좋고, 좋은 회사네!"
```

> **이 차이가 중요한 이유**: "착한 소비"는 1~2번 구매를 유도할 수 있지만,
> **"제품력"**만이 **재구매와 충성 고객**을 만든다.
> 동구밭은 이 두 가지를 모두 가진 드문 브랜드이다.

## 🧴 Part 1. 샴푸바 구매 페르소나

### 페르소나 1: 🧑‍💼 "두피 고민러" 민서 (32세, 직장인)

```
┌─────────────────────────────────────────────────────┐
│  이름: 김민서 (32세, IT 회사 마케터)                    │
│                                                     │
│  💇 두피 상태: 지성 + 가끔 트러블                       │
│  💰 월 뷰티 예산: 15~20만 원                           │
│  📱 정보 채널: 화해 앱, 유튜브, 인스타그램                │
│  🛒 구매 채널: 마켓컬리, 네이버 스마트스토어              │
│                                                     │
│  핵심 니즈: "두피가 시원하고 깨끗한 느낌"                 │
│  제품 선택: 동구밭 쿨링 샴푸바                           │
└─────────────────────────────────────────────────────┘
```

| 항목 | 상세 |
|------|------|
| **구매 계기** | 두피 트러블로 화학 성분 적은 샴푸를 찾다가 화해 앱 리뷰에서 발견 |
| **첫 인상** | "고체 샴푸? 이게 제대로 씻길까?" → 반신반의하며 시도 |
| **전환 포인트** | 거품이 풍성하고 두피가 실제로 개운해짐. 2주 사용 후 트러블 감소 체감 |
| **재구매 이유** | ① 두피 컨디션이 확실히 좋아짐 ② 펌핑 안 해도 되는 편리함 ③ 헹굼이 깔끔 |
| **사회적 가치 인식** | 3번째 구매 때쯤 "어, 이 회사 사회적기업이었어?" 하고 알게 됨 → 호감도 추가 상승 |

#### 민서의 구매 여정 (Customer Journey)

```
[인지]          [탐색]           [시도]          [전환]           [충성]
두피 트러블 → 성분 순한    → 반신반의로   → "이거 진짜   → 린스바까지
발생          샴푸 검색       첫 구매        좋은데?"        세트 구매
                                                          + 지인 추천
```

#### 민서의 핵심 한 마디
> "처음엔 그냥 두피에 순한 샴푸가 필요했어요. 써보니까 진짜 좋아서 계속 쓰는 건데, 나중에 보니까 좋은 일도 하는 회사더라고요. 그래서 더 기분 좋게 씁니다."

| **첫 인상** | "비누로 설거지가 된다고? 신기하네" |

| **재구매 이유** | ① 싱크대가 항상 깔끔 ② 세제 잔여물 걱정 없음 ③ 맨손 설거지해도 손 안 거칠어짐 |

```
[인지]           [탐색]         [시도]          [전환]           [충성]
싱크대 정리  → 고체 세제  →  "비누로     → "어? 이거     → 3개 묶음
하고 싶음      검색          설거지?"       진짜 되는데?"    정기 구매
```

#### 현우의 핵심 한 마디
> "자취방 싱크대가 좁은데, 세제 통이 없으니까 확 깔끔해졌어요. 그런데 세정력도 기대 이상이에요. 뽀득뽀득 닦이는 느낌이 액체 세제보다 오히려 낫다? 손도 안 거칠어지고요."

## 💡 종합 인사이트: "제품력 먼저, 가치는 보너스"

### 구매 결정의 진짜 구조

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   🧼 1단계: 제품력으로 문을 연다                               │
│   ────────────────────────────                              │
│   · 두피가 개운해진다 (샴푸바)                                 │
│   · 뽀득뽀득 깨끗하다 (설거지바)                               │
│   · 손이 안 거칠어진다 (설거지바)                               │
│   · 성분이 안전하다 (공통)                                     │
│                                                             │
│   ↓                                                         │
│                                                             │
│   💚 2단계: 가치가 "록인(Lock-in)"을 만든다                    │
│   ────────────────────────────────                          │
│   · "좋은 제품이 좋은 회사에서 나왔다"                          │
│   · 플라스틱 통이 안 나오는 뿌듯함                              │
│   · 장애인 고용에 기여한다는 기분 좋은 부가가치                   │
│                                                             │
│   ↓                                                         │
│                                                             │
│   🔁 3단계: 재구매 + 자발적 추천                               │
│   ──────────────────────────                                │
│   · "이 제품 진짜 좋아" → 먼저 품질 이야기                      │
│   · "게다가 좋은 회사야" → 자연스럽게 스토리 전달                │
│   · 동정이 아닌 존중 기반의 추천                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 마케팅 시사점

| 기존 접근 ❌ | 제안하는 접근 ✅ |
|-------------|-----------------|
| "장애인이 만든 착한 비누" | "진짜 좋은 비누, 진심 있는 사람들이 만듭니다" |
| "환경을 위해 구매하세요" | "써보면 알아요. 거품이 달라요" |
| "가치 소비를 실천하세요" | "두피가 까다로운 당신을 위한 샴푸바" |
| 동정심 자극 → 1회성 구매 | **제품 체험 → 품질 감동 → 가치 발견 → 충성 고객** |

### 핵심 메시지 제안

> **"먼저 좋아서 사고, 알고 보니 더 좋은 곳이었다."**
>
> 이것이 동구밭의 가장 강력한 브랜드 스토리다.
> 동정이 아니라 품질로 시작된 관계는, 가치로 더 깊어진다.
