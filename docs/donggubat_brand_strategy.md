## 1. 시장 분석

### 1-1. 동구밭이 속한 시장

동구밭은 **3개 시장이 교차하는 지점**에 위치합니다.

```
    ┌──────────────────┐
    │  비건·클린뷰티    │
    │  화장품 시장      │
    └────────┬─────────┘
             │
    ┌────────▼─────────┐
    │   ★ 동구밭 ★     │ ← 3개 시장의 교차점
    └──┬──────────┬────┘
       │          │
┌──────▼───┐ ┌───▼──────────┐
│ 제로웨이스트│ │ 사회적 기업   │
│ 생활용품   │ │ (장애인 고용) │
└──────────┘ └──────────────┘
```

| 시장 | 동구밭의 역할 |
|------|-------------|
| **비건·클린뷰티 화장품** | 고체 샴푸바, 스킨케어 등 제조 |
| **제로웨이스트 생활용품** | 설거지바, 고체치약 등 생활 전반 |
| **사회적 기업 / ESG** | 발달장애인 고용을 통한 사회적 가치 실현 |

---

### 1-2. 시장 규모 및 성장성

#### 🇰🇷 국내 비건 화장품 시장

| 연도 | 시장 규모 | 비고 |
|------|----------|------|
| 2013년 | 약 1,600억 원 | 초기 시장 |
| 2022년 | 약 5,700억 원 | 급성장기 |
| 2025년 | **약 1조 원** (전망) | 주류 시장 진입 |

- 국내 전체 화장품 시장: **약 17조 원** (2025년 기준)
- 비건·친환경 제품이 산업 성장을 견인하는 핵심 동력

#### 🌍 글로벌 고체 샴푸바 시장

- 연평균 성장률 (CAGR): **5~7%**
- 한국 헤어·두피 케어 시장: 2030년까지 연평균 **약 8.4%** 성장 예상

### 1-3. 시장 트렌드 (2024~2025)

#### 📈 5대 핵심 트렌드

| # | 트렌드 | 설명 | 동구밭과의 연관성 |
|---|--------|------|----------------|
| 1 | **제로웨이스트 가치 소비** | 플라스틱 용기 줄이기, 친환경 포장 | ✅ 핵심 정체성 그 자체 |
| 2 | **기능성의 세분화** | 탈모 완화, 두피 스케일링 등 고기능성 | ⚠️ 기능성 라인 확장 필요 |
| 3 | **성분 투명성 (클린뷰티 2.0)** | 파라벤·설페이트 배제, 천연 유래 성분 | ✅ 식물 유래 성분 + 이중 인증 |
| 4 | **스킨텔렉추얼 소비자** | 성분과 철학을 꼼꼼히 따지는 소비자 증가 | ✅ 투명한 브랜드 스토리 |
| 5 | **ESG 경영 필수화** | 기업의 사회·환경 책임이 구매 기준 | ✅ 장애인 고용 + 친환경 = ESG 모범 |

## 2. 무엇을 파는 브랜드인가?

### 2-1. 한 문장 정의

> **동구밭은 "플라스틱 없는 일상"을 파는 브랜드입니다.**  
> 고체 화장품과 생활용품을 통해, 나에게도 좋고 지구에도 좋은 지속 가능한 라이프스타일을 제안합니다.

---

### 2-2. 제품 카테고리 맵

```
                    동구밭 제품 세계
                         │
          ┌──────────────┼──────────────┐
          │              │              │
     🧴 헤어케어    🧼 바디·세안    🏠 생활용품
     ├─ 샴푸바       ├─ 바디워시      ├─ 설거지바
     ├─ 린스바       ├─ 클렌징바      ├─ 고체세탁세제
     └─ 트리트먼트바  └─ 입욕제       ├─ 고체치약
                                      └─ 욕실세제
          ┌──────────────┼──────────────┐
          │              │              │
     ✨ 스킨케어    🌸 퍼퓸        🎁 기업선물
     ├─ 팩클렌저     └─ 퍼퓸바      ├─ ESG 키트
     ├─ 앰플                       ├─ 사내 복지 세트
     └─ 선크림/선앰플               └─ 행사 기념품
```

---

### 2-3. 대표 제품 상세

| 제품 | 설명 | 가격대 | 판매처 |
|------|------|--------|-------|
| **🧴 샴푸바** | 두피 타입별 (지성·중건성·쿨링), 쑥·자몽 등 다양한 향 | 8,000~9,500원 | 공식몰, 올리브영, 쿠팡 |
| **💆 린스바/트리트먼트바** | 머릿결 관리, 샴푸바와 세트 사용 권장 | 8,000~9,000원 | 공식몰, 올리브영 |
| **🍽️ 설거지바 (워싱바)** | 1종 세제 (과일 세척 가능), 맨손 사용 OK | 약 4,500원/100g | 공식몰, 다이소 |
| **🧴 바디워시** | 고체 바디워시, 촉촉한 사용감 | 8,000~10,000원 | 공식몰, 올리브영 |
| **🪥 고체치약** | 플라스틱 튜브 대체 | - | 공식몰 |

### 2-4. 제품을 넘어 파는 것

동구밭은 **물건만 파는 것이 아닙니다.** 제품과 함께 3가지 가치를 판매합니다.

```
┌─────────────────────────────────────────────────────┐
│          동구밭이 실제로 파는 3가지                      │
├─────────────────┬─────────────────┬─────────────────┤
│   🧼 제품       │   🌍 라이프스타일 │   💛 참여 경험   │
│                 │                 │                 │
│  고체 화장품     │  플라스틱 없는    │  나의 구매가     │
│  생활용품       │  지속 가능한     │  발달장애인의    │
│  (기능적 가치)   │  일상 제안       │  일자리가 되는   │
│                 │  (환경적 가치)   │  경험 (사회적 가치)│
└─────────────────┴─────────────────┴─────────────────┘
```

> 📌 **소비자 입장에서:** "나는 비누를 사는 게 아니라, **더 나은 세상을 만드는 행동에 참여**하는 거예요."

## 3. 브랜드의 특장점

### 3-1. 5대 핵심 특장점

---

#### ① 🏭 대체 불가능한 제조 역량

> **한국 고체 비누 시장의 상당 물량이 동구밭 공장에서 생산됩니다.**

- 아모레퍼시픽, LG생활건강, 애경산업 등 대기업 OEM/ODM 수행
- 자체 브랜드 + 타사 제조까지 → **시장 생태계의 중심축**
- 경쟁사의 제품도 만들어주는 유일무이한 포지션

```
        동구밭 제조 역량 구조

   ┌─────────────────────────┐
   │    동구밭 공장 (제조)     │
   ├────────┬────────────────┤
   │ 자체 브랜드 │  OEM/ODM    │
   │  "동구밭"   │  대기업 납품  │
   │            │  경쟁사 제조  │
   └────────┴────────────────┘
         ↓           ↓
     B2C 판매     B2B 공급
   (공식몰,올리브영) (기업선물,납품)
```

---

#### ② 🏅 국내 유일 이중 인증

| 인증 | 발행 | 의미 |
|------|------|------|
| **🇫🇷 이브 비건 (EVE VEGAN)** | 프랑스 | 동물성 원료 배제 + 동물 실험 미실시 |
| **🇺🇸 USDA 유기농** | 미국 농무부 | 95% 이상 유기농 성분 + 전체 제조 공정 인증 |

> 🎯 **국내 최초이자 유일하게 두 인증을 동시 획득한 제조사**  
> → 글로벌 수준의 품질·안전성 신뢰 확보

---

#### ③ 💛 복제 불가능한 사회적 미션

- 전체 직원의 **50% 이상이 발달장애인**
- 매출 성장 → 장애인 추가 채용의 **자동 선순환 구조**
- '가꿈지기(발달장애 사원)' + '일굼지기(비장애 사원)'이 함께 일하는 문화
- 이 미션은 하루아침에 모방할 수 없는 **깊은 조직 문화**에 기반

> 📌 경쟁사가 제품은 따라 만들 수 있어도, **"함께 만드는 과정"**은 따라할 수 없습니다.

---

#### ④ 🌱 진정성 있는 친환경 실천

동구밭의 친환경은 마케팅 수단이 아니라 **기업 존재의 이유**입니다.

| 항목 | 실천 내용 |
|------|---------|
| **제품 제형** | 고체 제형 → 플라스틱 용기 사용 제로 |
| **포장재** | 비접착 재생 용지, 종이 포장 |
| **성분** | 식물 유래 100%, 화학 성분 최소화 |
| **동물 실험** | 완전 배제 (이브 비건 인증) |
| **환경 효과** | 매년 수백 톤 이상 플라스틱 절감 |

---

#### ⑤ 📊 검증된 비즈니스 성과

| 지표 | 성과 |
|------|------|
| **연 매출** | 100억 원 돌파 |
| **성장률** | 매년 50% 이상 |
| **투자** | 카이스트청년창업투자지주, 소풍벤처스 |
| **B2B 파트너** | 아모레퍼시픽, LG생활건강, 애경산업 |
| **유통** | 올리브영, 다이소, 쿠팡 등 주요 채널 입점 |

> 💡 "착한 기업은 돈을 못 번다"는 편견을 깬 **사회적 기업의 성공 모델**

## 4. 브랜드의 페르소나

### 4-1. 동구밭이 사람이라면?

> **동구밭은 "따뜻한 마음을 가진 실력 있는 이웃"입니다.**

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   👤 동구밭 페르소나                                      │
│                                                         │
│   이름: 동구밭                                            │
│   나이: 30대 초반 (2014년 탄생, 성장기)                     │
│   성격: 따뜻하지만 진지한, 말보다 행동으로 보여주는 사람       │
│   가치관: "좋은 일은 좋은 제품으로 증명해야 한다"             │
│   취미: 텃밭 가꾸기, 환경 다큐멘터리, 동료들과 함께하는 시간   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 4-2. 브랜드 성격 (Brand Character)

| 성격 키워드 | 설명 | 커뮤니케이션 예시 |
|-----------|------|----------------|
| **따뜻함** | 포근하고 정겨운, 마을 어귀의 안전한 공간 | "함께라서 더 좋은 오늘이에요" |
| **진정성** | 가식 없이 진솔한, 있는 그대로의 모습 | "우리의 비누에는 이야기가 담겨 있습니다" |
| **겸손한 실력** | 과시하지 않지만 확실한 품질 | "착하기만 한 게 아니라, 정말 좋은 제품이에요" |
| **포용** | 장애·비장애의 구분 없이 모두를 동료로 | "다르지만 함께, 그게 우리의 일상이에요" |
| **꾸준함** | 화려한 마케팅보다 묵묵한 실천 | "매일 만들고, 매일 함께 일합니다" |

|-------|---------|
| 담백하고 따뜻하게 | 과장되거나 화려하게 |
| 일상적인 언어로 자연스럽게 | 전문 용어를 나열하며 어렵게 |
| 제품의 기능을 솔직하게 | "착한 소비" 감성에만 호소 |
| 함께하는 이야기를 들려주듯 | 동정심을 유발하는 방식으로 |
| 고객을 동지(같은 가치를 공유하는 사람)로 | 고객을 단순 구매자로 |

#### 💬 커뮤니케이션 스타일 예시

```
❌ 이렇게 말하지 않아요:
"장애인이 불쌍해서 도와주세요. 우리 제품을 사주세요."

✅ 이렇게 말해요:
"우리는 함께 일합니다. 그리고 그 손으로 만든 이 비누,
  정말 좋아요. 한번 써보시면 알게 될 거예요."
```

### 4-4. 감성 키워드

```
    동구밭의 감성 세계

    ☀️ 따뜻한 아침 햇살
    🌿 텃밭의 흙 냄새
    🤝 동료와 나누는 미소
    📦 소박하지만 정성 가득한 포장
    🧼 거품 사이로 퍼지는 자연의 향

    ≠ 세련된 ≠ 럭셔리 ≠ 트렌디
    = 정겨운 = 진심인 = 함께하는 = 지속 가능한
```

### 4-5. 비주얼 아이덴티티

| 요소 | 특징 |
|------|------|
| **컬러** | 자연 친화적 톤 (어스톤, 그린, 베이지, 크래프트 브라운) |
| **이미지** | 인위적 연출보다 **일상 속 자연스러운 사용 장면** |
| **소재** | 재생 용지, 비접착 포장 — 소재 자체가 메시지 |
| **폰트** | Pretendard (한글, 깔끔하고 현대적) + Marcellus (영문, 클래식) |
| **점자** | 패키지에 **점자 표기** — 디자인이 곧 포용의 실천 |
| **사진 톤** | 내추럴 라이팅, 따뜻한 색감, 사람이 등장하는 컷 |

### 4-6. 브랜드 아키타입 (Brand Archetype)

동구밭의 브랜드 아키타입은 **"케어기버(Caregiver) + 시민(Everyman)"** 의 조합입니다.

```
┌──────────────────────────────────────────────┐
│           동구밭의 브랜드 아키타입               │
│                                              │
│   🤲 케어기버 (Caregiver)                     │
│   "세상을 돌보고 보살피는 자"                    │
│   · 발달장애인 동료를 돌보는 마음               │
│   · 지구 환경을 생각하는 책임감                 │
│   · 소비자의 건강한 일상을 위한 제품             │
│                                              │
│   🏡 시민 (Everyman)                          │
│   "모두와 함께하는 보통 사람"                    │
│   · 거창하지 않은 일상 속의 실천                │
│   · 특별하지 않아도 괜찮은 포용                 │
│   · 마을 어귀의 작은 텃밭 같은 친근함            │
│                                              │
│   → 결합: "평범한 일상에서 세상을 돌보는 이웃"    │
└──────────────────────────────────────────────┘
```

### 4-7. 브랜드 페르소나 한 장 요약

| 항목 | 내용 |
|------|------|
| **한 마디 정의** | "따뜻한 마음을 가진 실력 있는 이웃" |
| **핵심 성격** | 따뜻함, 진정성, 겸손한 실력, 포용, 꾸준함 |
| **말투** | 담백하고 따뜻하며 솔직한 |
| **절대 하지 않는 것** | 동정심 유발, 과대 광고, 화려한 치장 |
| **아키타입** | 케어기버(Caregiver) × 시민(Everyman) |
| **컬러 키워드** | 어스톤, 그린, 크래프트 |
| **감성 키워드** | 정겨운, 진심인, 함께하는, 지속 가능한 |
| **고객 호칭** | "동지" (동구밭을 아끼는 사람들) |
| **내부 호칭** | 가꿈지기(장애 사원), 일굼지기(비장애 사원) |
